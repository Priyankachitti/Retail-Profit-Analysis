# Retail-Profit-Analysis
An end-to-end data analysis project using Python and SQL to identify profit leakage in retail sales.
# Retail Profitability Analysis

## Project Overview
This project uses Python to analyze retail sales data, specifically focusing on identifying which product sub-categories are profitable and which are causing losses.

## Skills Demonstrated
* **Python Programming:** Data structures and logic.
* **Data Visualization:** Using Matplotlib to create business-ready charts.
* **Business Insights:** Identifying "Loss Leaders" in a retail environment.

## How to Run
1. Clone the repository.
2. Run `pip install pandas matplotlib`.
3. Execute `python profit_analysis.py`.

## Key Finding
Based on the analysis, products like **Tables** and **Binders** showed significant losses, suggesting a need for pricing adjustments.
